### ��️ 名前登録機能

### �� 注文機能

### �� 履歴機能

- 次回の食事に備える

## 使用方法

### 1. 名前登録
1. 「名前登録」タブを開く
2. 参加者の名前を入力
3. 「名前を追加」をクリック

### 2. 注文
1. 「注文」タブを開く
2. 注文者を選択
3. メニューを選択
4. 本数を設定
5. 「注文を追加」をクリック
6. 全員分の注文が完了したら「注文を確定」

### 3. 履歴確認
1. 「履歴」タブを開く
2. 店員さんへの注文内容を確認
3. 個人別の支払い額を確認

### 4. リセット
1. 「リセット」タブを開く
2. 「全てリセット」をクリック
